geoBoundaries: Open License

This page allows you to download the geoBoundaries Open product. The only requirement for use is acknowledgement. Users should be aware of the following usage notes for this product:

  • We seek to represent countries as they would represent themselves in the case of conflicted boundaries. This can result in overlaps between countries. If you need a single, seamless global boundary, please consider our CGAZ dataset.
  • Each boundary is the most recent, most accurate open license product we could create or source from sources with clearly defined licenses.
  • Every boundary has a unique ID across the entire globe.
  • You can choose between products with minor simplifications applied (the default below), or the original, highest-precision files (the "Large File Size") tab. While the differences in geometries are very small, the simplified geometries are frequently an order of magnitude (or more) smaller in file size and loading speeds. We recommend the simplified files for most uses.
